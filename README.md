# Hirelytic – AI Powered Resume and Interview Preparation Platform

## Overview

Hirelytic is an AI-powered career assistance platform developed for non-IT professionals, especially teachers and educational professionals. The system helps users build professional resumes, analyze resume strength, identify missing skills, match with suitable jobs, and prepare for interviews using Artificial Intelligence and Natural Language Processing (NLP).

The platform is designed to simplify the job preparation process by combining multiple career-support features into a single web application.

# Installation Steps

## Clone the Repository

```bash
git clone https://github.com/manjunath2003/Hirelytic---AI-powered-Resume-and-Interview-preparation-platform.git
```

## Move to Project Directory

```bash
cd Hirelytic---AI-powered-Resume-and-Interview-preparation-platform
```

## Create Virtual Environment

```bash
python -m venv .venv
```

## Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / Mac

```bash
source .venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
python app.py
```
